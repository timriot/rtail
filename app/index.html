Instantly stream terminal output to the browser with socket.io

Want to try rTail?

Add your streams to the live demo.

Install rtail and start streaming to the demo server – see the results above.

$ npm install -g rtail
$ echo "Hello from rtail.org demo" | rtail --id rtail.org --host rtail-demo.lukibear.com
⌘ + C

…or just start typing into stdin.

$ rtail --id rtail.org --host rtail-demo.lukibear.com --mute
⌘ + C
